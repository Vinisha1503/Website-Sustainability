# Website - Sustainability
A team-developed sustainability web application based on the UN Sustainable Development Goals, incorporating dynamic content loading and backend supporting interactive functionality

## Overview
This project is a team-developed web application based on the UN Sustainable Development Goals. It was designed to promote awareness of sustainability topics through an interactive and user-focused interface. Developed as part of a University Computer Science module, working collaboratively in a team.

## Features
- Dynamic content loading across multiple pages
- User authentication (login and account functionality)
- Structured navigation of sustainability topics
- Responsive frontend interface

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js (Express)
- Other: JSON for data handling
